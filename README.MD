# Simple Circuit: My First Zero Knowledge Proof Project

Welcome to my first project using Noir for generating zero knowledge proofs!

## Project Overview
This project demonstrates a basic circuit written in Noir, designed to help me learn the fundamentals of zero knowledge proofs (ZKPs).

## Key Learnings
- **Nargo.toml**: The manifest file for Noir projects. It defines project metadata and dependencies.
- **Binary Crate**: This project is a binary crate, meaning it compiles to an executable for generating ZK proofs.
- **Entry Point**: The main entry point is `src/main.nr`, which contains the `fn main()` function and a test function.
- **Libraries**: Noir supports libraries for code reuse. I plan to explore this feature in future updates.
- **File Extension**: Noir source files use the `.nr` extension.

## File Structure
- `Nargo.toml`: Project manifest and configuration.
- `src/main.nr`: Main circuit logic and tests.

## About Noir
Noir is a domain-specific language for writing zero knowledge circuits. It makes ZKP development accessible and efficient.
